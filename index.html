Hoi.

Ik ben Pepijn de Koning een Mediadeveloper. Ik ben vooral een front-end developer en heb ik heb een goed oog voor design.

Projecten.

Zona

Front & back-end

Zona was een project waar de opdracht was om een sociaal media platform te maken. Dit project heb ik in een team van twee gemaakt mijn taak was om het front-end gedeelte te maken en daarna te helpen met de back-end. Dit was het eerste grootte project dat ik moest maken met PHP en een data base. Ik heb er erg veel van geleerd en kan veel van deze functies toepassen in nieuwe websites.

Byellow

Front-end

Byellow is het laatste grootte project dat ik heb gemaakt. Dit project is ook gemaakt in team verband. Het doel was om een website te creëren voor een eigen media bedrijf. In dit project heb ik veel geleerd over animaties en meer out of de box denken. Ook heb ik geleerd hoe je goed met containers kan werken en veel wit ruimte kunt gebruiken om rust te creëren. Het gebruik van de kleuren was een keuze om op te vallen tussen alle Media bedrijven.
